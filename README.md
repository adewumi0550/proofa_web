# Proofa - AI Creative Workflow Certification

Proofa is the certification layer for AI-generated assets, allowing creators to monetize and protect their art while orchestrating powerful models.

## Requirements

Before you begin, ensure you have the following installed:

- **Node.js**: v18.17.0 or higher
- **npm**: v9.0.0 or higher (usually comes with Node.js)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adewumi0550/proofa_web.git
   cd proofa_web
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the App

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To create an optimized production build:

```bash
npm run build
```

To start the production server after building:

```bash
npm run start
```

## Project Structure

- `src/app`: Application source code and pages
- `src/components`: Reusable UI components
- `src/lib`: Utility functions
- `public`: Static assets
